# Справочник каталога деталей и материалов

## Назначение
Справочник содержит информацию о деталях, материалах "до внедрения", их нормах и единицах измерения, привязанных к цехам.

## Структура файла `catalog.xlsx`

### Колонки:

1. **Деталь (Part)** - код детали, например:
   - `КИР 03.614`
   - `Н 026.016.02`
   - `Н 026.116`

2. **Цех (Workshop)** - цех, к которому привязан материал:
   - `ПЗУ`
   - `ЗМУ`
   - `СУ`
   - `ОСУ`

3. **Тип позиции (Role)** - тип материала/операции:
   - `краска`
   - `разбавитель`
   - `отвердитель`
   - `лист`
   - `круг`
   - и т.д.

4. **Наименование материала 'до' (BeforeName)** - полное название материала или операции, который заменяется

5. **Ед. изм. (Unit)** - единица измерения:
   - `кг`
   - `шт`
   - `м`
   - и т.д.

6. **Норма (Norm)** - числовое значение нормы (формат: 0.0000)

7. **Примечание (Comment)** - опциональное поле для дополнительной информации

## Правила заполнения

- Каждая строка представляет одну позицию материала для конкретной детали в конкретном цехе
- Одна деталь может иметь несколько строк (например, краска, разбавитель, отвердитель)
- Один и тот же материал может быть в разных цехах - это разные строки
- Тип позиции (Role) определяет, что именно меняется в детали
- При заполнении программы, для каждой детали автоматически подтягиваются все строки из справочника
